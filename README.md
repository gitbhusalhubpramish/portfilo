## 📌 Project Overview

This is my personal portfolio website built using **Next.js App Router**, focused on:

* Clean UI and responsive design
* Modular and scalable structure
* Real backend integration (email system)
* Performance and modern deployment

---

## ⚙️ Tech Stack

* **Frontend:** Next.js (App Router), Tailwind CSS
* **Backend:** Next.js API Routes
* **Email Service:** Resend API
* **Deployment:** Vercel

---

## 🧠 How It’s Built (Core Concepts)

### 1. App Router Architecture

* Uses the **Next.js App Directory (`/app`)**
* File-based routing with server/client components
* Clean separation of UI and backend logic

### 2. Component-Based Design

* Reusable UI components
* Structured layout system
* Scalable folder organization

### 3. Styling with Tailwind CSS

* Utility-first styling
* Responsive design without writing custom CSS
* Fast UI iteration

### 4. Backend API Integration

* API routes inside:

  ```
  /app/api/contact/route.js
  ```
* Handles POST requests from the contact form
* Processes user input securely

### 5. Email System (Resend)

* Integrated **Resend API** for sending emails
* Server-side email handling
* Real-world production use case

### 6. Deployment (Vercel)

* Optimized for Next.js deployment
* Automatic builds and CI/CD
* Domain connected: **pramishbhusal.com.np**

---

## 📂 Project Structure

```
/app
  /api
    /contact
      route.js      # Backend email handler
  /components       # Reusable UI components
  /sections         # Page sections (About, Projects, etc.)
  layout.js         # Root layout
  page.js           # Main page

/public             # Static assets
/data               # JSON content
```
